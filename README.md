# 联想教育套装政策助手 - 天禧Claw Skill

## 功能
实时读取飞书多维表格，回答门店政策问题。数据实时同步，无需手动更新。

## 支持的问题示例
- "Y9000P 有什么政策？"
- "现在有什么活动？"
- "三件套怎么参加？"
- "教育补贴多少？"
- "晒单送什么？"
- "PO加磅多少钱？"
- "Y70手机有什么权益？"

## 安装方法
在天禧Claw中执行：
```
clawhub install lenovo-policy-assistant
```

## 数据来源
飞书多维表格（实时同步）：
- 活动作战面板：10条活动政策
- 产品速查表：567条产品数据

## 维护说明
由联想教育团队维护，更新飞书表格即可，门店无需重新安装。

# 联想教育套装政策助手 - 天禧Claw Skill

## 功能
联想门店政策顾问助手。查询当前活动政策、套装方案、教育补贴、晒单奖励、PO加磅、产品信息等。

## 安装方法
在天禧Claw中执行：
```
clawhub install zhangqingliang326/lenovo-policy-skill
```

## 使用示例
直接对天禧Claw说：
- "现在有什么活动？"
- "Y9000P 有什么政策？"
- "三件套怎么参加？"
- "教育补贴多少？"
- "晒单送什么？"
- "PO加磅多少钱？"

## 文件说明
- `SKILL.md` — Skill 定义文件（告诉AI怎么回答）
- `knowledge-base.md` — 政策知识库（活动政策+产品数据）
- `README.md` — 本文件

## 数据更新
联系维护人员更新 knowledge-base.md 文件即可。
